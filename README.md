# Консалтинг по международной аккредитации школ

Минималистичный лендинг на русском языке для консультационных услуг Дулата Иржанова по аккредитации школ по стандартам CIS и IB.

## Структура проекта
- `index.html` — основная страница.
- `styles.css` — стили с монохромной стеклянной эстетикой.
- `script.js` — логика отправки почтового запроса.
- `assets/favicon.svg` — монохромный значок сайта 32×32.
- `_redirect-helper/services.html` — вспомогательный редирект для размещения подподраздела.
- `LICENSE` — лицензия MIT.

## Локальный просмотр
Откройте `index.html` в браузере или поднимите статический сервер:

```bash
python -m http.server
```

Затем перейдите по адресу `http://localhost:8000/`.

## Публикация
Проект оптимизирован для GitHub Pages и для размещения в подпапке основного домена. После включения GitHub Pages с веткой `main` сайт будет доступен по адресу `https://dulatirzhanov.github.io/services/`.
